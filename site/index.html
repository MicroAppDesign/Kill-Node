Kill Node.js

เคยไหมที่ทำงานอยู่ดีๆ เครื่องค้างจากการรัน Node.js เบื้องหลังจำนวนมากโดยไม่รู้ตัว? กว่าจะรู้ตัวว่าเครื่องอืดหรือค้างเพราะ process ที่กินทรัพยากรอยู่...

ดาวน์โหลดจาก GitHub

ปัญหาที่กวนใจ

หากคุณกำลังเจอปัญหานี้ แล้วไม่รู้จะจัดการยังไง หรือไม่อยากเปิด Task Manager เพื่อไล่หา process ที่ต้องการหยุด?

ทางออกของคุณ

kill-node คือเครื่องมือที่ช่วยให้คุณหยุด Node.js processes บน Windows ของคุณได้อย่างง่ายดายและรวดเร็วทันใจ!

ออกแบบมาเพื่อ Local

เครื่องมือขนาดเล็กที่เน้นการใช้งานบน Windows ในเครื่องของคุณโดยตรง ไม่ต้องติดตั้งอะไรเพิ่มเติม

ส่วนประกอบในโปรเจค

  • kill-node.bat — สคริปต์แบบ Batch สำหรับ Windows
  • kill-node.ps1 — สคริปต์แบบ PowerShell สำหรับ Windows

วิธีใช้งาน

เปิด PowerShell ในโฟลเดอร์โปรเจคของคุณแล้วพิมพ์คำสั่ง:

.\kill-node.ps1

หรือถ้าต้องการใช้ Batch file:

.\kill-node.bat

คุณสมบัติการเลือกจัดการ

  • รายการ Node.js process จะแสดงขึ้นมาให้เลือกชัดเจน
  • เลือกหมายเลขเพื่อปิดเฉพาะ Process นั้นๆ
  • เลือก A เพื่อปิดทั้งหมด (Kill All)
  • เลือก Q เพื่อออกจากโปรแกรม
ออกแบบมาเพื่อพัฒนาประสบการณ์การเขียนโปรแกรมบนเครื่อง Local ของคุณ
