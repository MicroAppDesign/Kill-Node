Kill Node.js

เครื่องมือเล็ก ๆ สำหรับ Windows ที่ช่วยให้คุณจัดการ Node.js process ได้ง่าย ๆ โดยไม่ต้องเปิด Task Manager หรือจำ PID เอง

Purpose

สคริปต์นี้ออกแบบมาเพื่อหยุด Node.js process ที่ค้างอยู่หลังการทดสอบหรือรันงานที่ไม่ได้ปิดเอง

Target

Windows users ที่ต้องการ clean-up Node.js workflow แบบเร็ว และควบคุมได้ไม่ kill ทุก process โดยไม่ตั้งใจ

Deploy

พร้อมใช้ GitHub Pages ด้วย workflow ที่ deploy โฟลเดอร์ site อัตโนมัติเมื่อ push ไปยัง main

What is included

  • kill-node.bat — Batch script สำหรับ Windows
  • kill-node.ps1 — PowerShell script สำหรับ Windows
  • site/index.html — หน้า presentation สำหรับ GitHub Pages
  • .github/workflows/pages.yml — deploy workflow

How to use

เปิด PowerShell แล้วไปยังโฟลเดอร์โปรเจค จากนั้นรันคำสั่งต่อไปนี้:

.\kill-node.bat

หรือใช้ PowerShell script:

.\kill-node.ps1

Kill process แบบเลือกได้

ทั้งสองสคริปต์จะ:

  • แสดงรายการ Node.js process ที่กำลังรัน
  • ให้ผู้ใช้เลือกหมายเลขเพื่อ kill เฉพาะ process ที่ต้องการ
  • ให้ผู้ใช้เลือก A เพื่อ kill all หรือ Q เพื่อยกเลิก
  • จะแสดงผล และรอให้กด Enter ก่อนปิดหน้าต่าง

Quick commands

# ใน PowerShell
.\kill-node.bat

# หรือ
.\kill-node.ps1

Why this repo

Repo นี้ช่วยให้การจัดการ Node.js process บน Windows เข้าใจง่ายและปลอดภัยขึ้น โดยเฉพาะเมื่อคุณต้องหยุด process ที่ค้างอยู่หลังการพัฒนา หรือทดสอบแบบอัตโนมัติ

This repo is designed for humans: easy to read, easy to use, and inspiring to share.
